# Accessibility (a11y) Improvement Design

> Date: 2026-03-23
> Status: Approved
> Standard: WCAG 2.1 AA

## Overview

WCAG 2.1 AA 감사 결과 15개 이슈 발견. 모든 컴포넌트에 키보드 네비게이션, 스크린 리더 지원, 색상 대비, ARIA 레이블 적용.

## Findings: 3 Critical, 7 Major, 5 Minor

### Critical
1. file-uploader.tsx: 파일 입력에 접근 가능한 레이블 없음 (4.1.2)
2. url-input.tsx: 입력 필드에 레이블 없음 (3.3.2)
3. converter-page.tsx: 에러 메시지가 입력과 연결되지 않음 (3.3.1)

### Major
4. header.tsx: 드롭다운 키보드 접근 불가 (2.1.2)
5. header.tsx: 모바일 메뉴 포커스 관리 없음 (2.1.1)
6. footer.tsx: 링크 색상 대비 부족 (1.4.3)
7. markdown-preview.tsx: 코드 블록 레이블 없음 (1.1.1)
8. tool-card.tsx: 포커스 인디케이터 없음 (2.4.7)
9. download-button.tsx: 복사 상태 알림 없음 (4.1.2)
10. converter-page.tsx: 에러-입력 연결 없음 (3.3.1)

### Minor
11. url-input.tsx: 비활성 버튼 대비 (1.4.11)
12. seo-content.tsx: FAQ 시맨틱 구조 (1.3.1)
13. file-uploader.tsx: 드롭존 role 없음 (1.3.1)
14. header.tsx: 로고 aria-label (1.1.1)
15. 번역 파일 a11y 키 추가
